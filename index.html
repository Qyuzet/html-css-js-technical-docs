Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language. Inside a host environment (for example, a web browser), JavaScript can be connected to the objects of its environment to provide programmatic control over them.

JavaScript is commonly used to create dynamic web pages.

The language is also used in environments like Node.js to develop server-side applications.

It is a popular language because of its simplicity and versatility.

let message = "Hello, JavaScript";
What you should already know

This guide assumes you have the following basic background:

  • A general understanding of the Internet and the World Wide Web (WWW).
  • Good working knowledge of HyperText Markup Language (HTML).
  • Some programming experience. If you are new to programming, try one of the tutorials linked on the main page about JavaScript.

Understanding HTML and CSS is also helpful when learning JavaScript.

const pi = 3.14;
JavaScript and Java

JavaScript and Java are similar in some ways but fundamentally different in others.

Both Java and JavaScript use similar syntax, but they are used for entirely different purposes.

Java is a compiled, object-oriented programming language.

JavaScript, however, is interpreted and runs in web browsers.

console.log("JavaScript vs Java");
Hello world

To get started with writing JavaScript, open the Scratchpad and write your first "Hello world" JavaScript code:

This will help you understand the basics of the language.

function greetMe(yourName) { alert("Hello " + yourName); } greetMe("World");
Variables

You use variables as symbolic names for values in your application.

A variable can be defined using the `var`, `let`, or `const` keywords.

let x = 10;
Constants

Constants are block-scoped, much like variables declared using the `let` keyword.

Once assigned a value, a constant cannot be reassigned.

const MY_CONSTANT = 100;
